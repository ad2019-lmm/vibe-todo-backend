# TODO Backend

Node.js 기반 TODO 애플리케이션 백엔드 서버

## 시작하기

### 설치

```bash
npm install
```

### 실행

```bash
# 개발 모드 (nodemon 필요)
npm run dev

# 프로덕션 모드
npm start
```

서버는 기본적으로 `http://localhost:3000`에서 실행됩니다.

todo-backend/
